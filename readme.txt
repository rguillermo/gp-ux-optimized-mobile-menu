Convert the standard GP menu to a UX optimized float menu.

== Description ==

GP UX Optmized Mobile Menu is thinking for mobile users. Detach the standard menu and put it to the bottom right
with a UX optimized desing.

Currently is only tested in GeneratePress Theme.

is written in pure javascript without the use of libraries, so it is very light and has no impact on page loading time.

It is also responsive to all devices even when resizing the screen.

== Installation ==


1. Upload the gp-ux-optimized-mobile-menu.zip to the `/wp-content/plugins/` directory
2. Decompress it
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
